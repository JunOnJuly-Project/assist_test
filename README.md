# 온디바이스 AI 에이전트 프로젝트 (On-Device Code Agent)

본 저장소는 프라이버시가 100% 보장되는 완전 로컬 환경에서, NVIDIA GPU가 없는 **노트북(인텔 CPU-Only)** 제약을 뚫기 위해 시스템 자원을 지독할 정도로 최적화한 초경량 AI 시스템입니다. 
7~8B 규모 오픈소스 모델과 효율성이 극한인 파이썬 Code-Agent 샌드박스를 결합해 RAG 기능을 가동합니다.

## 📁 디렉토리 구조 가이드

코드는 메모리와 보안 격리 원칙에 따라 아래와 같이 분리되어 동작합니다.

*   `backend/`: FastAPI 오케스트레이터 및 백그라운드 RAG 태스크. CPU 과열을 막기 위한 **Thread-Lock(통신 락)** 모듈과 보안 가드레일 제어기 스크립트.
*   `frontend/`: 상태 모니터링 시각화를 담당하는 Next.js 14+ / shadcn 대시보드 (실시간 CPU 스로틀 및 RAM 점유 표기).
*   `sandbox/`: 파이썬 코드를 쏟아내는 에이전트의 액션을 OS 디렉토리 침범 없이 격리 실행하는 **`uv run --isolated`** 제로 트러스트(Zero-Trust) 공간.
*   `database/`: Vector 정보와 Entity 그래프가 저장되는 초경량 DB(LanceDB) 폴더.
*   `weights/`: `Qwen2.5-7B-Coder` 등 다운로드한 CPU 기반 GGUF 가중치 모델이 보관되는 오프라인 벙커 (Git Tracker 금지됨).
*   `docs/`: 프로젝트 관리를 위한 블루프린트, 인프라, 워크플로우 명세와 평가 보고서 묶음.

## 🚀 시작하기 및 필수 검문 기준 (Phase 0 ~ 1)

**🚨 주의사항 (5 TPS 이상 합격 기준)** : 노트북의 CPU-Only의 치명적 한계를 감당하려면 응답 생성 속도(TPS)가 필수적입니다. 코딩 시작 전 아래 터미널 구동을 확인하십시오.

1.  **Ollama 세팅 및 TPS 측정**: `ollama run qwen2.5-coder:7b` 등을 실행하여 CPU 연산 속도가 **"초당 최소 5토큰 이상"** 매끄럽게 나오는지 확인 후 백엔드를 준비합니다. (실패 시 3B 급 모델인 `Phi-3.5-mini`로 강등 전개하십시오).
2.  **격리 샌드박스(uv)**: OS에 Rust 기반 파이썬 초고속 패키지 관리자인 `uv` 환경 경로가 등록되었는지 확인합니다.
3.  **Backend 준비**: `cd backend` 경로 진입 후 `uv`를 통해 의존성을 구축하고 Phase 1 API 개발을 준비합니다.
